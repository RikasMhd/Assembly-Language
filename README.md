This repository contains Assembly Language Programs (ALP) written for learning and practice purposes.It is useful for students, beginners, and exam preparation who want to understand low-level programming concepts.
About Assembly Language:Assembly Language is a low-level, processor-dependent programming language that uses mnemonics to represent machine instructions.It provides direct control over hardware and helps understand how a CPU works internally.

                                                            -------Questions---------
SumInt:
Read two integers and print their total. 
Note: Do not print any messages for reading inputs or for the output.

23 56
79

Exercise 1:

Create three integer variables num1, num2 and num3.
Read three numbers and store them in these variables.
Find the sum of the values in these three variables and print it.

Exercise 2
Read 5 numbers and store them in an array.

Exercise 3
a. Read an integer n and print all the numbers from 0 to n
  Input: 5
  Output: 0, 1, 2, 3, 4, 5
  
b.. Read two integers n1, n2 and print all the numbers from n1 to n2
  Input n1: 5
  Input n2: 10
  Output: 5, 6, 7, 8, 9, 10
  
c. Read three integers n1, n2, n3 and print all the numbers from n1 to n2 incremented by n3
  Input n1: 5
  Input n2: 10
  Input n3: 3
  Output: 5, 8
